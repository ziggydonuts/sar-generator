# 🕵️‍♂️ Co-Investigator AI: Agentic AML Compliance

## 📌 Project Overview
**Co-Investigator AI** is a conceptual multi-agent framework designed to automate **Suspicious Activity Report (SAR)** generation. It solves the high cost and low scalability of manual Anti-Money Laundering (AML) workflows.

---

## ⚙️ Proposed System Architecture
Our design uses a **Multi-Agent Orchestration** approach to ensure accuracy:

* **Data & Privacy:** Ingests case data through a **Privacy Guard** and **RBAC Agent** for secure masking.
* **Intelligence:** A **Retrieval Agent (RAG)** gathers external evidence to ground the narrative.
* **Detection:** Uses **K-means clustering** and pattern analysis to mitigate false positives.
* **Synthesis:** A **Planning Agent** coordinates specialized agents to draft the SAR.
* **Validation:** A **Compliance Validation Agent** reviews the output before human approval.
